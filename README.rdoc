=Spotify Playlist Rater

This app uses Spotify's web API to rate users' playlists on a hipster scale.

{Delayed Job}[https://github.com/collectiveidea/delayed_job] is used for handling scoring playlists.

The {RSpotify gem}[https://github.com/guilhermesad/rspotify] is used for communicating with Spotify.

Pusher[http://pusher.com/] is used for live updating recently scored playlists.
